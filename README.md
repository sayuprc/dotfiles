- agent (AI Agent: GitHub Copilot CLI, Claude Code CLI)

- git
- mise
- starship
- vim
- vscode

## セットアップ

### 前提条件

[mise](https://mise.jdx.dev/) をインストールしてください。

```bash
curl https://mise.run | sh
```
